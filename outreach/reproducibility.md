# Reproducibility

## 120-180 word message

I am sharing a reproducibility-focused project that may be of interest to people who study scientific software, benchmark design, or audit trails for computational work. The project is a revival and empirical audit associated with Bombelli's 1987 causal-set simulated-annealing code/program. It ports the historical program to Python 3.12 and adds small inputs, reproducible CPU experiments, ensemble summaries, diagnostics, and citation metadata.

The emphasis is on auditability rather than broad scientific claims. The repository makes it possible to rerun the small examples, compare schedules and seeds, inspect diagnostics, and cite a fixed public release. It does not claim to solve causal set theory, quantum gravity, or the causal set embedding problem.

The project may be useful as a compact example of how older scientific code can be revived, documented, and archived in a way that supports careful reuse and criticism.
