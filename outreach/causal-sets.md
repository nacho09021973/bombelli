# Causal Sets

## 120-180 word message

I am sharing a small project that may be of interest to researchers and readers around causal set theory and historical computational physics. It is a revival and empirical audit associated with Bombelli's 1987 causal-set simulated-annealing code/program for studying small causal-set embedding examples. The repository includes a Python 3.12 port, benchmark inputs, repeated CPU runs, order-theoretic diagnostics, and a citable Zenodo archive.

The project is intentionally limited in scope. It does not claim to solve causal set theory, quantum gravity, or the embedding problem. Its purpose is to make an early computational approach inspectable with current tools and to document what can be observed when the revived program is run repeatedly under controlled settings.

I would be grateful for corrections, historical context, or suggestions about framing. The work is best read as a reproducible software artifact and audit trail, not as a new theoretical claim.
